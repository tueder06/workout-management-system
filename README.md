# Workout management system

Mobile application allowing users to organize and track their workouts.  

- User authentication: sign up and log in.  
- Create and manage workout plans divided into days and exercises.  
- Users can choose exercises from a predefined list or create custom exercises.  
- Start workout sessions with timers for sets and rest periods, automatically calculating total weight lifted and total workout time at the end.  
- Track progress with statistics and a monthly calendar view.  
- Manage user profile (personal data and account settings).  
- Integration with Samsung wearable devices (smartwatch).
